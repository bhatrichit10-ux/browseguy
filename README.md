# browseguy
my own very little browser which is fully cli and made in Node.JS
## Install
```
npm i -g browseguy
browseguy
```
# Dependencies
- node-fetch
- cheerio
- html-to-text

## Features
- CLI only
- Can be used to naviagte
- Color formatting
